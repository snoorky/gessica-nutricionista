# Géssica Nutricionista

Site destinado a Géssica Carvalho, uma das mais renomadas nutricionista da cidade de Sorocaba. Nesse site foi desenvolvido com o intuito de fazer seus pessoas que estão buscando ter uma vida saudável ou pretendem alcançar algum objetivo e não sabe qual rumo tomar. Além disso, o site possui um sistema de livro de receitas, na qual o usuário consegue consultar algumas receitas, pesquisar pelo ingrediente, pela categoria da alimentação ou até mesmo restrição alimentar. 

## Funcionalidades

- Artigos de recursos nutricionais
- Livro de receitas
